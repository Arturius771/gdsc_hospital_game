MediGame
-An app for Solution Challenge 2022. Targetting UN Goal 3. This app will help children understand the hospital environment. 

-Video: https://www.youtube.com/watch?v=onJ80RkCDMs

Authors: 
-David Cooke
-Paul Cotarelo
-Artur Foden

Software: 
-Flutter/Dart for the demo available here


How to run:
-Please clone this repository into an IDE such as Visual Studio Code or Android Studio. 
-You will need to install Flutter as per https://docs.flutter.dev/get-started/install

-Please then build and run the medi_game_app/lib/main.dart file.
